# CameraPhotoTools（AI生产）

## 相机拍摄照片后的一些处理小工具
目前仅有windows版
#### raw
目前包含cr2,cr3,nef,nrw,arw,srf,sr2,dng,erf,mef,mos,orf,pef,ptx,pxn,raf,raw,rw2,rwl,srw,x3f,3fr,fff,eip,bay。
欢迎补充
#### jpg
包含.jpg和.jpeg

## 功能
1. 查找指定目录下的jpg对应指定目录下的raw，不存在对应的raw则删除
2. 查找指定目录下的raw对应指定目录下的jpg，不存在对应的jpg则删除
3. 批量对目录下的jpg文件重命名
